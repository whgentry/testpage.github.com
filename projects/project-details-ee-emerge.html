EE Emerge - Junior Design Project

Collaborated with a team of four engineers to build a custom interactive electronic system that appealed to the public. We built a 3D version of tic-tac-toe that is playable using capacitive touch sensors and LEDs. I developed the C software, I2C communication, LED control, capacitive sensor design, state machine, and game algorithm. As a team we implemented microcontrollers, sensors, and LEDs into a two-foot interactive cube with a playable version of 3D tic-tac-toe.
